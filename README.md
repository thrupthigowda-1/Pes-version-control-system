# Pes-version-control-system

## Objective
To implement a simplified Git-like version control system similar to Git.

## Features
- Blob, Tree, Commit object storage
- SHA-based hashing
- Index (staging area)
- Commit system
- Log/history tracking

## Phases
- Phase 1: Object storage
- Phase 2: Tree structure
- Phase 3: Index handling
- Phase 4: Commit system

## Output
All integration tests passed successfully.

## How to Run
make clean
make pes
./test_sequence.sh
